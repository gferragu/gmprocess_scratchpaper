SETUP
  • The software is provided in a *.zip file containing a directory "bc" with all necessary MatLab figure and function files. Unzip it where you want it.
  • BCseis was working under MatLab 2017b.
  • The "help_path" variable  in Line 71 in BCseis.m should be changed to reflect the location of the help webpages in your local directory structure.
  • Invoke the program by typing "BCseis" (no quotes) in the MatLab command window.

That's it.
